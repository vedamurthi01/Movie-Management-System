# 🎬 Movie Management System

## 📖 About the Project

Movie Management System is a web-based application designed to manage movie information efficiently. It allows users to browse movie details and administrators to add, update, delete, and manage movie records through a simple and user-friendly interface.

## ✨ Features

* 👤 User Registration and Login
* 🎥 Add New Movies
* ✏️ Update Movie Details
* 🗑️ Delete Movies
* 🔍 Search Movies
* 📂 Categorize Movies by Genre
* ⭐ View Movie Ratings
* 📱 Responsive User Interface

## 🛠️ Technologies Used

* ☕ Java
* 🌱 Spring Boot
* 🗄️ MySQL
* 🎨 HTML
* 🎨 CSS
* ⚡ JavaScript
* ⚛️ React.js

## 🎯 Project Objective

The objective of this project is to provide an efficient platform for managing movie records and enabling users to easily search and view movie information.

## 🚀 How It Works

1. Users log in to the system.
2. Administrators can add, edit, and delete movie records.
3. Users can browse and search movies.
4. Movies are displayed with details such as title, genre, rating, and release date.
5. Data is stored and managed securely in the database.

## 🌟 Benefits

* Easy movie record management.
* Fast search and retrieval of movie details.
* Organized movie catalog.
* User-friendly interface.
